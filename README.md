# Landing Page Kasarao Hamburgueria
Landing Page para Hamburgueria

Este projeto é uma landing page moderna e responsiva para uma hamburgueria, desenvolvida com foco em performance, design atrativo e experiência do usuário. A página apresenta os principais produtos, promoções e informações de contato de forma simples e visualmente impactante.

Ideal para estudos de front-end, portfólio ou uso comercial, o projeto utiliza boas práticas de desenvolvimento web e possui estrutura organizada para fácil personalização.

Funcionalidades
Layout responsivo para desktop e mobile
Seção de destaque para hambúrgueres e combos
Cardápio visual com imagens e preços
Botão de contato/WhatsApp
Navegação simples e intuitiva
Tecnologias Utilizadas
HTML5
CSS3
JavaScript
Objetivo

Criar uma experiência visual moderna para divulgar uma hamburgueria de forma profissional e atrativa.
